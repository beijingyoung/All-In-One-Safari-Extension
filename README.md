## 开启 Safari 隐藏调试菜单

退出 Safari

开启终调试菜单，终端录入：

	$ `defaults write com.apple.Safari IncludeInternalDebugMenu 1`

关闭终调试菜单，终端录入：
	
	$ `defaults write com.apple.Safari IncludeInternalDebugMenu 0`
	

注： 5.1 与 5.2 功能不同。
